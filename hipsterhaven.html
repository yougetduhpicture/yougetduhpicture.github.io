Hipster Haven

  • Time: Spring 2021
  • Team: Just me
  • My role: Programming, Service Design, Back-end, Front-end
  • Tools: Eclipse, MariaDB, PuttyIO

Assignment Description:

We had an assignment to create a service with a back-end, that uses MariaDB database that contains the required tables and a few example objects on the table. We also had to create the front-end for the service.

The Requirements:

  • Search, add, change, delete information from database
  • use control level servlet classes for handling the http-responses made with DAO-classes utilising JDBC-interface
  • The web app could be either be done using JSP. in server or javascript/Jquery… on browser
  • The app must not crash even if it gets wrong input.
  • One table is enough for the database

My Database:

I made a fictional music album listing service called Hipster Haven.

One could search for albums and artists. The table included the following information: Artist name, Album name, Cover art (image as a link), release year.

The database was a MariaDB hosted by Haaga-Helia(each student has their own) I also made the website structure that showed and listed the information.

What I learned:

I learned how to make databases, how to use, putty, SQL, how to create modify, and delete data from a database. Probably also a lot more such as general coding skills increasing and creative thinking.

Feedback:

My project filled all the requirements. It had a basic good UI and it worked. Teacher also liked the fact I had stored album covers on the database.
